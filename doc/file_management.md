# 文件处理类

## 1. 简介

file_management 这个类主要用于对文件的管理，检测文件的参数等

## 2. 接口

### file_management

1. 检测文件等类型

### aac_file 

1. 检测文件是否是aac文件
2. 获取aac文件的参数，包括
   1. 文件路径
   2. 是否是aac文件
   3. 音频时长，单位：秒
   4. 采样率
   5. 声道数
   6. 比特率
   7. 封装格式，包括：ADTS，ADIF
   8. 编码配置，包括：AAC LC，HE-AAC(v1)，HE-AAC(v2)，AAC LD，AAC ELD
   9. 帧长度
3.
